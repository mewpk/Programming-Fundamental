ประกาศตัวแปรจำนวนเต็ม ได้แก่ answer , x, y

แสดงข้อความ "Enter first number :"

รับค่าจากแป้นพิมพ์เก็บที่ตัวแปร x

แสดงข้อความ "Enter second number :"

รับค่าจากแป้นพิมพ์เก็บที่ตัวแปร y

ทำซ้ำๆไปเรื่อยๆ เงื่อนไขคือ x != y
    ถ้า x > y
        ให้ x -= y
    ถ้าไม่ตรงเงื่อนไข
        ให้ y -= x
ให้ answer = x 
แสดงข้อความ ตัวแปร answer
