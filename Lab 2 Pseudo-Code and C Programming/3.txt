ประกาศตัวแปรจำนวนเต็ม ได้แก่  number
แสดงข้อความว่า "Enter number :"
รับค่าทางแป้นพิมพ์เก็บที่ตัวแปร number
ให้ i = 1
ทำซ้ำไปเรื่อยๆ โดยที่เงื่อนไขคือ i <= number
    ทำซ้ำไปเรื่อยๆ โดยที่เงื่อนไขคือ j <= number
        ถ้า i == 1 หรือ i == number
            แสดงข้อความว่า "*"
        ถ้าไม่ตรงเงื่อนไข
            ถ้า j == 1 หรือ j == number
                แสดงข้อความว่า "*"
            ถ้าไม่ตรงเงื่อนไข
                แสดงข้อความว่า " "
        ให้ j++
    แสดงข้อความว่า "\n"
